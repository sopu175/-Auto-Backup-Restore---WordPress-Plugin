== Description ==
This plugin allows you to backup and restore your WordPress plugin directory with one click.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate it through the WordPress admin panel.

== Features ==
* One-click backup of plugins directory.
* Restore any backup.
* Delete unwanted backups.
